## : 작업단계

1. JIRA에서 TODO 작업 생성 및 담당자 할당
2. 디자이너가 IN DESIGN으로 옮긴 후 작업
3. 작업 완료된 피그마 링크를 작업내용에 작성 후, DESIGN COMPLETE로 이동

4. 작업 완료 후 Pull Request 만들기
